## 🤖 About Dev Nexus

**Dev Nexus** is a feature-rich Discord bot built specifically for **Roblox developer communities**. Whether you're running a development studio, managing a group, or building your first game — Dev Nexus has everything you need to keep your server organized and your team productive.

### ✨ Features

- 🎮 **Roblox Lookups** — Search games, users, assets, and groups using the Roblox API
- 📋 **Role Applications** — Full application system with reviews, interviews, approvals and rejections
- 🧑‍💼 **Developer Portfolios** — Create and discover developer profiles with skills, showcase games, and availability
- ⭐ **Vouch System** — Vouch for developers and build reputation with ratings and a leaderboard
- 🛠️ **Dev Tools** — Luau snippets, API lookups, DevForum search, and live Roblox service status
- 💸 **DevEx Calculator** — Calculate exactly how much USD you'd earn from cashing out Robux
- 📊 **Game Compare** — Compare two Roblox games side by side with live stats
