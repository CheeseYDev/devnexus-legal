🔒 Privacy

Privacy Policy

Last updated: March 19, 2026  •  Dev Nexus Discord Bot

✅ Dev Nexus does not sell your data. We only store what is necessary to provide the bot's features.

1. Introduction

This Privacy Policy explains what data Dev Nexus ("the Bot", "we", "us") collects, how it is used, and how it is stored. By using the Bot, you agree to the collection and use of information as described in this policy.

2. What Data We Collect

We collect only the minimum data necessary to operate the Bot's features:

Data Type What We Store Why
Discord User ID Numeric ID only To identify users for portfolios, vouches, and to-do lists
Discord Username / Tag Username at time of vouch To display vouch attribution
Portfolio Data Roblox username, role, skills, bio, links User-submitted portfolio information
Vouch Data User ID, comment, rating, timestamp Vouch and reputation system
To-Do Items Task text and completion status Personal task management feature
Application Responses Proof text or image URLs submitted during applications Role application review system
Server Configuration Channel IDs, category IDs, role IDs Bot setup and configuration per server

3. What We Do NOT Collect

  • Message content outside of direct interactions with the Bot
  • Passwords, email addresses, or payment information
  • IP addresses or device information
  • Data from channels the Bot is not actively used in

4. How We Use Your Data

All collected data is used solely to provide the Bot's features. Specifically:

  • Portfolio data is used to display developer profiles when requested
  • Vouch data is used to build reputation scores and leaderboards
  • Application responses are stored temporarily for staff review
  • Server configuration data is used to route the Bot's features correctly

We do not use your data for advertising, analytics, or any purpose beyond operating the Bot.

5. Data Storage

All data is stored in JSON files on our hosting server. Data is stored indefinitely until you request deletion or use the Bot's built-in deletion commands (such as /portfolio delete). We take reasonable steps to protect stored data but cannot guarantee absolute security.

6. Third-Party Services

The Bot interacts with the following third-party services to provide its features:

  • Roblox API — to fetch game, user, asset, and group data. Governed by Roblox's Privacy Policy.
  • Discord API — to operate within Discord. Governed by Discord's Privacy Policy.

We do not control how these third parties handle data and are not responsible for their privacy practices.

7. Data Deletion

You can delete your data at any time using the following commands:

  • /portfolio delete — removes your portfolio
  • /vouch remove — removes a vouch you gave
  • /todo clear — clears completed tasks

To request full deletion of all your stored data, please contact us through our Discord server and we will remove it manually within 7 days.

8. Children's Privacy

The Bot is not directed at children under the age of 13. We do not knowingly collect data from children under 13. If you believe a child under 13 has provided us data, please contact us so we can remove it.

9. Changes to This Policy

We may update this Privacy Policy from time to time. We will update the "Last updated" date at the top of this page when changes are made. Continued use of the Bot after changes constitutes acceptance of the updated policy.

10. Contact

If you have questions or concerns about this Privacy Policy or want to request data deletion, please reach out through our Discord server.
